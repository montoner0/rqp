v1.0.0.586
+ Добавлена поддержка M3U и PLS при проигрывании URL. Также, в качестве бонуса, для таких случаев в заголовке плеера показывается адрес реального потока.
+ Добавлена экспериментальная фича по обновлению панели при переходе к следующему/предыдущему файлу. Таким образом плеер увидит изменения, которые произошли в фоне
  (например, можно запустить в фоне копирование и сразу начать прослушивание, новые файлы будут сразу доступны).
  Теоретически, фича может вызвать замедление при переходе по трекам на "медленных" панелях.
